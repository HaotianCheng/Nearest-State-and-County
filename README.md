# Nearest-State-and-County
Get Nearest State and County By Entering Latitude and Longtitude

## Installation (linux)
Make sure your current folder is Nearest State/src/Project

To compile the file:
```bash
javac /Project/Draw.java
```
To run the file:
```bash
java /Project/Draw
```
